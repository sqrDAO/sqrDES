# Brand Assets

Logo and brand assets for the sqrDAO ecosystem are hosted on the official site. Download assets from:

**[sqrdao.com/brand-kit](https://www.sqrdao.com/brand-kit)**

Available formats: **SVG** and **PNG** for all sub-brands (sqrDAO, sqrFUND, sqrVOICE, sqrNODE, sqrBUIDL, Web3 Builders' Summit) in dark and light variants.

This repo does not duplicate assets — use the official brand kit as the canonical source.
